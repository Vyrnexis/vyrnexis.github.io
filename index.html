Deploy

Create a repository called vyrnexis.github.io and push this file as index.html to the root of the main branch.

git init
git add index.html
git commit -m "Initial Vyrnexis site"
git branch -M main
git remote add origin https://github.com/<your-username>/vyrnexis.github.io.git
git push -u origin main
